WallPaper App

Project information

  • Category: WallPaper
  • Client: Practice Project, Self
  • Project date: 02 January, 2019

Simple wallpaper app that works with Pixel, Unsplash API to fetch list of wallpapers. We can download the wallpaper or we can share it with friends. The wallpapers are according to the Screen resolution of the mobile app. App has also divided the wallpaper into the category and user can also search the wallpaper according to their designs. Unfortunately it was for my practice so it never got to the playstore.

  • Download Wallpaper
  • Select Categories of wallpaper.
  • Set wallpaper to lock screen.
  • Share Wallpaper
  • Favorites
  • Show Wallpaper information.
  • Set wallpaper to main screen.
