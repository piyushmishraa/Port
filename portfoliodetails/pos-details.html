POS App

Project information

  • Category: Sales
  • Client: Local Business, Pakistan
  • Project date: 02 August, 2020
  • Project URL: https://play.google.com/store/apps/details?id=com.oditar.oditarpos

Point of sale application for android tablets for an Arabic client and I am still working on it. The App is basically intended to use by waiters and the managers and Chefs to check orders and for multiple purposes. The App can work offline and has an ability to share data on Close network with devices that are only connected to Lan not WAN. Using this app waiter can register orders on the spot for any tables and customers and have the features a POS have and is backward compatible with its other options with Database migration.

  • Login
  • Configure server.
  • Make Orders
  • Work Offline
  • Multilingual
  • Print orders.
  • Configure Tax rates.
  • Configure item prices.
